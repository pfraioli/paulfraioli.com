# paulfraioli.com

Static personal website for Paul Fraioli.

The site is intended to be hosted on Cloudflare Pages at:

https://www.paulfraioli.com/

## Files

- `index.html`: crawlable profile page with metadata and JSON-LD structured data
- `styles.css`: responsive page styling
- `assets/hero-abstract-lithograph-emerald-softline-v2.png`: active hero background image with deeper green accent and softened pen line
- `assets/hero-abstract-lithograph-emerald.png`: earlier hero background image with saturated green accent
- `assets/hero-abstract-lithograph.png`: earlier generated abstract lithograph hero option
- `assets/hero-minimal-green.png`: earlier generated minimalist hero option
- `assets/hero-minimal-red.png`: earlier generated minimalist hero option
- `assets/hero-geopolitics.png`: earlier generated hero background option
- `publications/strange-days-ai-next-industrial-revolution.pdf`: hosted featured publication
- `robots.txt`: crawler access and sitemap pointer
- `sitemap.xml`: canonical URL for search engines
- `llms.txt`: machine-readable summary for AI agents and retrieval systems
